# TextCraftRS

## What the program does

- Reads a text file line by line  
- Removes extra spaces between words  
- Trims leading and trailing spaces  
- Writes the cleaned text into `output_clean.txt`


## Example

**Before:**

Hello     world
This   is    a   test

**After:**

Hello world
This is a test

## Features

- Cleans and formats text files
- Removes extra spaces between words
- Trims leading and trailing whitespace
- Saves cleaned output to `output_clean.txt`
- Fast and efficient (written in Rust)

## How to Use

Download the binary below and place your input file in the same folder.

Run the tool like this:

./texttidy input.txt


The cleaned output will be saved as:

output_clean.txt
